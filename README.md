*Plataforma multi-tenant para el sector gastronómico que permite a múltiples restaurantes gestionar menús digitales mediante QR dinámicos desde un único panel centralizado[cite: 1].*
* 🔗 **Demo en Vivo:** [Panel Administrativo Menú Digital](https://admin-menu.onrender.com/WebCony/Login)[cite: 1]

  * **Pasarela de Pagos:** Integración completa de Mercado Pago (Checkout Pro) con sincronización mediante Webhooks en tiempo real, garantizando el 100% de la trazabilidad financiera[cite: 1].
  * **Internacionalización (i18n):** Arquitectura global con detección automática de idioma y cambio dinámico sin recarga de página desde un archivo de traducción centralizado[cite: 1].
